# Project Plan

## Project Title
OSINT Political Narrative Analysis

## Case Focus
Public opinion and political narratives related to a political or social event, based on publicly available social media data.

## Main Objective
To analyze how people discuss, interpret, and emotionally respond to political or social events in digital environments.

## Research Questions
- What are the dominant narratives in social media discussions?
- What emotions appear most frequently in the discourse?
- How do users frame key actors, events, and conflicts?
- Can NLP and machine learning help classify emotions, narratives, or political stance?

## Planned Workflow
1. Dataset selection
2. Data cleaning
3. Exploratory data analysis
4. Keyword and topic analysis
5. Emotion and narrative labeling
6. Baseline machine learning classification
7. Dashboard development
8. Analytical report writing

## Planned Labels

### Emotion Labels
- Anger
- Fear
- Trust
- Hope
- Neutral

### Narrative Labels
- Security threat
- Regional stability
- Humanitarian concern
- Diplomatic solution
- Identity based narrative
- Unclear or mixed

### Stance Labels
- Supportive
- Oppositional
- Neutral
- Ambiguous

## Expected Outputs
- Cleaned dataset
- Exploratory notebook
- NLP analysis notebook
- Baseline machine learning model
- Streamlit dashboard
- Analytical report
